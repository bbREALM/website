## Fusion Project File and Illustrated Tutorial  

- This design was created using AutoDesk Fusion, with the corresponding project format being .f3d. This is a software that can be used for free for personal use; you can download it and import the file to fully understand my creation steps and modify the model. Since this was initially provided to sponsors, you can download it after sponsoring.  
- Because this is a more challenging project, I also wrote an illustrated tutorial for the sponsors at that time.
